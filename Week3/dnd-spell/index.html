Great Gamble

This spell can be used once per 2 long rests and can be used instead of rolling dice for any D20 check.

The user says "Alright- check this out!" outloud (both in and out of character).

After doing so, they must flip 5 coins.

  • If there are more heads than tails, then the result of the D20 check is a natural 20.
  • If there are more tails than heads, then the result of the D20 check is a natural 1 with double the consequences.
Upon obtaining the latter of the outcomes, the user must also click the button below.
